このソフトにできること

用意したシナリオでウェブページをダウンロードする。

テキストファイルで書かれたシナリオに沿って、ウェブページやファイルをダウンロードするソフトです。 ウェブ検索のトップ１００件を１つのテキストファイルにまとめたり、一括ダウンロードするといった作業を、コマンドで指示できます。 フォーム送信にも対応しますので、投稿などに使えます。 ログイン、閲覧や投稿、ログアウトといった一連のシナリオを用意して、全自動で実行させることができます。 個別ページの解析は外部スクリプトとして切り離しているので、拡張や修正が容易。 簡単なスクリプトを書くだけで様々なページに対応できます。 VBScript だけでなく、あらゆる言語で書いたスクリプトやプログラムを組み合わせて利用できます。 VBScript だけで書かれているので、ほとんどのウィンドウズ環境でそのまま動きます。 ライセンスがフリーなので、好き勝手にカスタマイズできます。

動作環境

  • Windows 2000 以降すべて。
  • Windows 2000 の場合、以下のコンポーネントが無ければ、 マイクロソフトのページからダウンロード して追加してください。
    • MSXML バージョン6以上
    • MDAC バージョン2.5以上

配布しているファイルについて

  1. ZIP形式で圧縮しています。 Windows 2000 以前では、この形式を解凍できるソフトウェアが別途必要です。
  2. 圧縮ファイルには以下のファイルが含まれます。
    wpost.vbsソフトウェア本体。
    他の *.vbs付属のシナリオ見本で使うページ解析スクリプト。
    yahoo.batヤフー検索を一発で行うバッチの見本。
    w_keyword_temp.txtyahoo.bat が使う作業ファイルの見本。
    テンプレ.txtシナリオの見本。シナリオの書き方説明もある。Bing検索を行うシナリオ。
    他の *.txtグーグル検索、ヤフー画像検索、ヤフーログインなどのシナリオ見本。
    ライセンス.txtこのソフトウェアのライセンスです。
    初めにお読みください.htmlこの解説書です。
    その他解説書の使う画像ファイルなど。

ご使用の手引き - とりあえず使ってみる

  1. シナリオ見本は C:\tmp を使う前提で書いています。
    • これは変更できますが、それにはシナリオの書き方を覚えないといけません。
    • とりあえず動かしてみるなら、 C:\tmp, C:\tmp\bing, C:\tmp\y, C:\tmp\g を作ってください。
  2. 解凍したファイルの テンプレ.txt を読み、何が起きるか知っておいてください。
  3. テンプレ.txt アイコンを wpost.vbs アイコンにドロップします。
  4. シナリオが開始されます。終了までしばし待ちます。
  5. 終了合図のダイアログが出ます。
  6. C:\tmp\bing にできたファイルと テンプレ.txt の内容を見比べて、何が起きたか理解してください。

ご使用の手引き - 詳しく知りたい - wpost コマンドの実行

  1. wpost.vbs は、マウスでシナリオファイル（指示書）をドロップすれば、そのシナリオを実行します。
  2. コマンドラインから引数でシナリオファイルを渡して実行することもできます。
  3. 複数のシナリオを受け取ることができます。
  4. コマンドラインオプションで、シナリオファイルの文字コードを指定できます。 例 /e:shift_jis
  5. 文字コードを指定しないときは utf-8 として扱います。

ご使用の手引き - 詳しく知りたい - yahoo 一発検索コマンドを試す

  1. ファイルを解凍したフォルダで、コマンドプロンプトを開きます。
  2. yahoo エクセル　開発
    というように、 yahoo に続いて、検索したい言葉をスペース区切りで入れてエンターを押します。
  3. 検索結果が画面に出ます。
  4. 画面が流れて読みきれないと思いますが、 C:\tmp\y\w_result_temp.txt に保存されています。

ダウンロードしたプログラムが動かないとき

  • 助けを求める前に、セルフチェックで確認しましょう。
  • スクリプトが動かない人のセルフチェッカー
